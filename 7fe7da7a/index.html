jsDelivr - 一个免费、快速、可靠的开源CDN
发布时间: 2020-04-20, 下午

CDN的全称是Content Delivery Network，即内容分发网络。CDN是构建在网络之上的内容分发网络，依靠部署在各地的边缘服务器，通过中心平台的负载均衡、内容分发、调度等功能模块，使用户就近获取所需内容，降低网络拥塞，提高用户访问响应速度和命中率。CDN的关键技术主要有内容存储和分发技术。——百度百科

官网

jsDelivr

介绍

jsDelivr 是用于开源文件的免费 CDN。与 Github 和 npm 紧密集成，能够自动为几乎所有开源项目提供可靠的 CDN 服务。提供稳定的 CDN，可用于大量流量的热门网站上生产。没有带宽限制或高级功能，它完全免费供任何人使用。一切都经过优化并不断改进，为所有用户提供最大的速度和正常运行时间。

使用

通过 jsDelivr 引用资源 https://cdn.jsdelivr.net/gh/user/repo@version/file
使用特定版本

https://cdn.jsdelivr.net/gh/jquery/jquery@3.2.1/dist/jquery.min.js

使用版本范围而非特定版本

https://cdn.jsdelivr.net/gh/jquery/jquery@3.2/dist/jquery.min.js

https://cdn.jsdelivr.net/gh/jquery/jquery@3/dist/jquery.min.js

使用最新版本

https://cdn.jsdelivr.net/gh/jquery/jquery/dist/jquery.min.js

添加 .min 到任何 JS/CSS 文件后获取缩小版本，如不存在将自动生成

https://cdn.jsdelivr.net/gh/jquery/jquery@3.2.1/src/core.min.js

合并多个文件，组合格式 /combine/url1,url2,url3

https://cdn.jsdelivr.net/combine/gh/jquery/jquery@3.1/dist/jquery.min.js,gh/twbs/bootstrap@3.3/dist/js/bootstrap.min.js
